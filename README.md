# ASEEIoT
Example code for ASEE ThingSpeak Workshop
The send code example writes a temperature value to ThingSpeak from the ESP8266 and a MPL3115A2 temperature sensor
  the wiring is ESP82666->MPL3115A2
                GND------->GND
                3V3------->VCC
                PIN2------>SDA
                PIN14----->SCL
